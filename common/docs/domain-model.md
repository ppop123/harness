## 业务规则（Business Rules）

> 这些规则是不变的业务约束。代码必须遵守，AI 必须理解。

```
BR-001: [规则描述]
         例：一个用户同时只能有一个活跃订阅

BR-002: [规则描述]
         例：订单金额不能为负数

BR-003: [规则描述]
         例：已完成的订单不可取消
```
